# 🛒 WoBasar

**WoBasar** ist ein moderner Online-Marktplatz speziell für Studierende.
Die Plattform ermöglicht es, unkompliziert Anzeigen für Produkte und Dienstleistungen zu erstellen, zu durchsuchen und miteinander zu interagieren.

Dieses Projekt entsteht im Rahmen der Vorlesung **Verteilte Systeme** als Teamarbeit.

---

## ✨ Features

* 🔍 **Anzeige durchsuchen** (z. B. Bücher, Elektronik, WG-Zimmer)
* ➕ **Eigene Anzeigen erstellen**
* 💬 **Nachrichten zwischen Nutzern**
* ❤️ **Merkliste für Favoriten**
* 📂 **Kategorisierte Inhalte**
* 🌙 **Dark Mode / Light Mode**
* 📱 **Responsive Design (Mobile & Desktop)**

---

## 🧱 Tech Stack

* **Frontend:** Blazor
* **Backend:** REST-API
* **UI Framework:** MudBlazor
* **Sprache:** C#

---

## 🚀 Getting Started

### Voraussetzungen

* IDE (z. B. Visual Studio / VS Code)

### Installation

```bash
git clone https://github.com/juliba1/WoBasar
cd wobasar
dotnet restore
dotnet run
```
